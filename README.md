# Компонент CommentsSection

Компонент отображения и добавления комментариев с оптимистичным UI.

## Возможности

- **Оптимистичное обновление**: комментарий отображается в списке сразу после отправки
- **Асинхронная отправка**: используется `addCommentAction`, имитирующий POST-запрос
- **Обработка ошибок**: при ошибке комментарий удаляется, выводится сообщение
- **Очистка формы**: поле ввода очищается при успешной отправке
- **Хуки**: реализовано с использованием `useOptimistic` и `useActionState` (React 18+)

## Используемые хуки

- `useOptimistic` — для предварительного отображения нового комментария
- `useActionState` — для обработки формы и асинхронного действия без ручного `preventDefault`

## Пример использования

```jsx
import { CommentsSection } from './components/CommentsSection';

function App() {
  return <CommentsSection />;
}
```

## Поведение

1. Пользователь вводит комментарий и нажимает "Отправить"
2. Комментарий немедленно появляется в списке с пометкой `(отправка...)`
3. Если отправка прошла успешно — комментарий закрепляется
4. Если возникла ошибка — комментарий исчезает, отображается ошибка


### Примечание

Компонент требует React 18+ и совместимости с хуками `useOptimistic` и `useActionState`
Не работает в старых версиях React или при ручной обработке `submit`
